文件结构：
	db.h - 数据库模块
	menu.h - 对话菜单模块
	tablemaker.h - 表格模块
	school.h - 学生、老师、课程数据存储结构和相关函数
	school.cpp - 主程序
	config/ - 菜单文件和配置文件

Linux平台下，g++-4.7.3、clang-3.3编译通过；Cygwin平台下，g++
4.8.3、clang-3.4.2编译通过。
因为涉及了中文的输入、输出，所以使用了wchar_t存储中文数据。wchar_t的大小在不同平台不一样，Linux下是4字节，Windows下是2字节。所以存储的数据不一定能跨平台使用，已经将不同平台区别开来。
文件都是UTF-8编码，只在Linux和Cygwin下测试。因为没有Visual
Studio，所以不知道是否会有乱码。
另外读菜单和配置文件时是读"config/xxx"，所以可执行程序要在config目录的上一级运行。
如果编译或者使用有问题，可以发回邮件询问对应情况。花了很多时间写，还是不想让一些奇怪的原因影响成绩的TAT。
